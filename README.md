# vue3-antvX6-draggable

1. 基于 vue3.x 和 antv X6 制作一款可视化拖拽组件 具有高扩展性 自定义组件等强大功能
2. 在线体验：https://cuizeyu.github.io/vue3-antvX6-draggable/

## 本地运行这个项目(node18 以上)

1. `npm i` 安装依赖
2. `npm run dev` 等编译完成后本地打开页面看效果。

x6: https://x6.antv.antgroup.com/
